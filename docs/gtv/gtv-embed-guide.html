How to embed G|TV videos in your website

如何在您的網站中嵌入G|TV的視頻

Copy the following HTML code into your page, the videoid parameter is the ID of the video you want to embed.
複製下面的HTML代碼到您的頁面中，用videoid 參數指定視頻的ID。 


    <iframe width="560" height="315" src="https://ourhimalayas.github.io/gtv-embed/?videoid=5f9d838fad3f0f61d2a6736b"
        allowfullscreen frameborder="0"
        allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture">
    </iframe>
